# Commands used for AES Encryption

## 1. CBC Mode
- Encryption
```console
openssl enc -aes-128-cbc -e -in test.txt -out cipher-cbc.bin -K 9876543210 -iv 0123456789
```

- Decryption
```console
openssl enc -aes-128-cbc -d -in cipher-cbc.bin -out decrypted-cbc.txt -K 9876543210 -iv 012345678
```
